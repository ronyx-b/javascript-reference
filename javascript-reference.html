Syntax, Keywords and Declaration

Basic Syntax

// …line comment. /* … */block comment.

(…) grouping operator for expressions … , … Comma operator: evaluates each, returns last.

Variables Declarations

var variable = value, … declares a global scope variable, may be initialized to value, may declare an initialize a set of comma-separated variables.

let variable declares a block level variable.

const constant declares a constant value o reference to an object.

Functions

function funct(parameters, …) { …; return output } function declaration: may receive "parameters", runs containing statements when called, may "return" the specified "output" if given, else "undefined" is implied.

var funct = function functName(parameters, …) { …; return output } function expression: assigns the given function to the variable; may be named, but only referred in the expression (useful when self-recurring).

(parameters, …) => { …; return output; } arrow function notation: creates anonymous functions, if it has only a single statement then its result is taken implicitly as return. arguments functions' built-in array-like object that contains the arguments list. .name nonstandard property that holds the function's name (even named function expressions). .call(thisArg, arguments, …) calls a function and sets its "this" reference to "thisArg". .apply(thisArg, argArray) calls a function and sets its "this" reference to "thisArg", provides arguments for the functions as and array "argArray". .bind(thisArg, arguments, …) creates a new function that, when called, has its "this" keyword set to the provided "thisArg".

funct(arguments, …) function calling: invokes a defined function, may require arguments to be passed.

(functionStatement)(arguments) Immediately invoked function expression: immediately calls the function defined in the grouping operator (it is a grouping operator and an invoking operator).

Compound Data types var array = [ values,… ] array literal declaration: declares a variable "array" initialized with "values". array[index] access the element at position "index" of the array. ...list spread/rest syntax: spreads an array to a list where a list is expected or condenses a list into an array. var object = { property: value, …, method: function(…) {…} } object literal declaration: can contain properties, methods and other objects, declared as "key: value". delete keyword operator that deletes a property from an object. [a, b,…] = [x, y,…]; {a, b,…} = {a:x, b:y,…} Destructuring assignment: allows you to assign the properties of an array or object to variables using syntax that looks similar to array or object literals.  Operators Arithmetic Operators + sum | string concatenation - subtraction * multiplication / floating point division ** exponentiation % modulus (integer division remainder) ++ increment (prefix or postfix) -- decrement (prefix or postfix) oper= assignment, simple or compound (may be preceded by other operands: +=, -=, *=, /=) Comparison Operators == equal value != not equal value === equal value and type !== not equal value or not equal type > greater than < less than >= greater than or equal to <= less than or equal to …?…:… ternary operator, evaluates first operand and if true returns the one left from the colon, if false the right one =…||… conditional assignment, if the first value is "falsy" (null, undefined, false), then assign the second value Logical Operators && returns true if both statements are true || returns true if any one statement is true ! reverses the result Type operators typeof variable returns a string with the type of "variable" (string, number, boolean,…). object instanceof class returns true if an object is an instance of an object type. property in object returns true if "property" (as a string) is a member of "object" (object[property] != undefined). void expression evaluates the given expression and then returns "undefined". Logical Structures and Loops Flow Control: Break and Continue break label; declares an exit of the innermost loop or the one defined by the "label". continue label; declares a jump to the next iteration of the innermost loop or the one defined by the "label". If Else Statements if (condition) { …; } if condition evaluated is true, executes contained statements. else if (condition) { …; } if previous conditions are false and the one indicated is true, run script. else { …; } if all previous conditions are false, run script. Switch Case switch (statement) { cases… } evaluates the result of "statement" and executes the "case" that matches. case value: …; break; executes statements if "statement" is equal to "value", ends with a "break". default: script; break; run "script" if "statement" is not equal to any defined case value. Do While Loops label: while (condition) { …; } executes statements while defined "condition" is true. label: do { …; } while (condition) executes statements once and as long as "condition" is true. For Loops label: for (initial; condition; step) { …; } sets the "initial" statement, then runs "script" while "condition" is true, executing the "step" statement at the end of each iteration. for (key in object) { …; } executes statements for each "key" property in the "object", using "key" as iterator key string (name of the current parameter of the object, refer as "object[key]"). for (item of iterable) { …; } executes statements for each "item" in the "iterable" object (array or array-like, string) using "item" as iterator.  Error Handling, Promises and Modules. Error Handling try { … } defines a block of statements that might produce an error. throw expression defines an exception thrown, can be a string or an "Error" object (new Error(msg)). catch(error) { … } defines a block of statements to handle an error produced on a "try" block, and stores it in the "error" object of type "Error()". .message property of "Error" type objects that contains the error message. .name property of "Error" type objects that contains the error name. finally { … } optional final block of statements that will be executed weather or not the "try" block threw an error. Promises (Asynchronous Functions) return new Promise(function callback(resolve, reject) { …; } ) the function won't return an immediate value, but rather will either be successful and send the value in a "resolve(…)", or fail and send an error message from a "reject(…)". resolve(resolved) function declared in promise's callback that returns a "resolved" promise. reject(rejected) function declared in promise's callback that returns a "rejected" promise. .then(function callback(resolved) { … }) Promise's method that takes the "resolved" value and executes the statements that would follow, they can be chained (asyncFn(…).then(…).then(…).catch(…)). .catch(function callback(rejected) { … }) Promise's method that takes the "rejected " value and executes the statements that would follow. .finally(function callback() { … }) Promise's method that is called at the end of a promise chain when the promise is settled, whether fulfilled or rejected. async function asyncFn(parameters,…) { … } declares an asynchronous function that returns an implied promise, can be used inside "try …catch" blocks. await asyncStatement used only inside "async" functions, stops its execution until a resolved or rejected promise is returned from the "asyncStatement" and returns that value. Modules import exported as alias,… from 'module-or-file-path' declares a resource imported (static) from another module, where the resources "exported" can be: the default module "defaultExport", all "*" (can also be done as "import 'module'"), or a specific list "{ exportedN as aliasN, … }". import('module') declares a dynamic import when called as a function and returns the module as a "Promise". export default resource declares a "resource" (function, class, object, variable) to be exported to other modules, where only one can be set as "default" which doesn't need to be named. Built-in Objects, Properties, Methods and Functions Parent Object Object(…) constructor function for new objects, equivalent to: var newObject = {}; can be initialized with a list of comma-separated pair of values in the form of “property: value,”, where value can variables, functions or other objects. .create(objectTemplate) class (static) method that creates a new object with the same properties of "objectTemplate" through inheritance (with same initial values by default, stored in ".prototype"). .getPrototypeof(object) class (static) method returns the prototype of the specified object. .toString() returns a string representing the object. String Objects Methods String(…) constructor function for strings, can convert from numbers. .length property that returns the length of the string. .charAt(index) strings method, returns character at "index". .concat(string) receives a string to be concatenated at the end of the string it is applied. .includes(string) returns true if the "string" argument is contained in the applied string. .startsWith(string) returns true if the string applied starts with the given "string", otherwise false. .endsWith(string) returns true if the string applied ends with the given "string", otherwise false. .match(regExp) returns the matches of the applied "regExp" regular expression. .toLowerCase() returns the string with all characters in lower cases. .toUpperCase() returns the string with all characters in upper cases. .trim() removes blank spaces from both ends of the string. .replace(regExp, string) returns the string with the matches of the "regExp" replaced by "string". .search(regExp) returns the index of the first match of the "regExp", or "-1" if not found. .slice(start, end) returns a new string from "start" index (inclusive) until "end" index (not inclusive) if given, or until the end of the original string; indexes can be negative (from the end of the string) .split(separator, limit) returns an array of substring from the original string divided by a "separator" string or regular expression, but until reaching a "limit" number of elements if given. .substr(start, length) (deprecated) returns a portion of the original string, from "start" index and with a defined by "length" if given. .substring(start, end) strings method, returns part of a string starting a "start" index (inclusive) and ending before "end" index (not inclusive, end of string if omitted). .charCodeAt(index) string method, returns Unicode value of the character at "index". Array Objects Properties and Methods Array(…) constructor function for arrays, empty if no arguments, otherwise with its arguments as the values. .isArray(argument) "Array()" class (static) method that returns true if the argument is an array. .length property of any array (and array-like objects like strings and arguments list) that returns their length. Array Modifying Methods: .push(element) adds the arguments to the end of the array and returns the next index. .unshift(element) adds the arguments to the start of the array and returns the new length. .pop() removes the last element of the array and returns it. .shift() removes the first element of the array and returns it. .concat(elements) returns the original array joined together with other arrays or values provided in the arguments. .slice(start, end) returns a new array from "start" index (inclusive) until "end" index (not inclusive) if given, or until the end of the original string; indexes can be negative (from the end of the string). .splice(start, deleteCount, elements…) changes the contents of an array from "start" index by removing (if "deleteCount" > 0) and/or adding new "elements" (if any) in place, and returns an array with the removed items (empty if none). .fill(value, start, end) assigns "value" to the elements of the array, from "start" index (inclusive, default "0") to "end" index (not inclusive, default "array.length"). .reverse() reverses the order of the elements of an array in place (the original array). Array Query Methods: .indexOf(element, start) returns the index of the first element from "start" index (inclusive, default "0") that corresponds to "element", or "-1" if not found. .lastIndexOf(element, start) returns the index of the last element from "start" index and going backwards (inclusive, default "array.length - 1") that corresponds to "element", or "-1" if not found. .incluides(element) returns "true" if the array includes the given element, otherwise "false". .join(string) returns a string created by concatenating all elements in the array with the given "string" delimiter. .toString() returns a string representing the array and its elements. . toLocaleString() returns a localized string representing the array and its elements. Array Iterating Methods: .forEach(function callback(element, index, array) {…}) takes a callback function as argument that is called upon each "element" of the "array". .map(function callback(element, index, array) { …; return mapElement; }) takes a callback function as argument that is called upon each "element" of the "array" and returns a new array out of the returned values of the callback. .find(function callback(element, index, array) { …; return boolean; }) returns the first element for which the "callback" function returns "true". .findIndex(function callback(element, index, array) { …; return boolean; }) returns the index of the first element for which the "callback" function returns "true". .filter(function callback(element, index, array) { …; return boolean; }) returns a new array containing all elements for which the "callback" function returns "true". .every(function callback(element, index, array) { …; return boolean; }) returns "true" if all of the elements in the array return "true" on the callback function. .some(function callback(element, index, array) { …; return boolean; }) returns "true" if at least one of the elements in the array returns "true" on the callback function. .reduce(function callback(accumulator, element, index, array) { …; return toAccumulator; } , initialValue) executes a reducer function on each element assigning its returned value to the given "accumulator", and returns the final value of it; can have an "initialValue" set for the "accumulator". .reduceRight(function callback(accumulator, element, index, array) { …; return toAccumulator; } , initialValue) executes a reducer function on each element, going from last to first, assigning its returned value to the given "accumulator", and returns the final value of it; can have an "initialValue" set for the "accumulator". .sort(function callback(first, second) { …; return number; }) sorts the original array, by converting to strings and comparing their Unicode value, or by result of "callback" comparing two elements and sorting "first" before "second" if it returns a "number" <= 0, or backwards otherwise. Date Methods Date(…) constructor function for date objects, by default (if no arguments) to the current time, otherwise accepts arguments in order (year, month zero index, day, hour, minutes, seconds) .now() "Date" class (static) method that returns the number of milliseconds elapsed since the "Unix Epoch" (January 1, 1970 00:00:00 UTC). .toLocaleString() returns the instantiated date in local string format. .getTime() returns the number of milliseconds of the instantiated date since the "Unix Epoch". .getMonth() returns the month of the instantiated date with zero index (0 for January – 11 for December). .getDate() returns the day of the month of the instantiated date (1 – 31). .getDay() returns the day of the week with zero index of the instantiated date (0 for Sunday – 6 for Saturday). .getFullYear() returns the four-digit year of the instantiated date. .getHours() returns the hour in 24-hour format (0 – 23) of the instantiated date. .getMinutes() returns the minutes of the instantiated date. .getSeconds() returns the seconds of the instantiated date. Math Methods Math Built-in object that contains mathematical operation methods and constants. .PI property that contains the value of "pi" (π ≈ 3.1416). .E property that contains the value of Euler's constant (e ≈ 2.7183). .abs(number) returns the absolute value of the "number" argument. .max(numbers) returns the greatest value among the parameters .min(numbers) returns the lowest value among the parameters .pow(base, exp) returns the result of the "base" number elevated to the "exp" power. .exp(exp) returns the result of the power of "e" elevated to the number "exp". .log(number) returns the natural logarithm (base "e") of the given "number". .sqrt(number) returns the square root of the parameter. .ceil(number) returns the parameter rounded up. .floor(number) returns the parameter rounded down. .round(number) returns the parameter properly rounded. .random() returns a number at random between 0 (inclusive) and 1 (not inclusive); that is: [0,1) .toFixed() method to fix the number of decimals for a float to amount in argument or zero if not specified. .sin(x); .cos(x); tan(x); .asin(x); .acos(x); .atan(x); .asinh(x); .acosh(x); .atanh(x); .asinh(x); .acosh(x); .atanh(x); trigonometric and hyperbolic functions, angle units are in radians. Console Methods console object with methods that display information on the console. .log(…) general output of logging information. .warn(…) outputs a warning message. .error(…) outputs an error message. .table(…) displays tabular data as a table. .dir(object) displays an interactive list of the properties of the specified JavaScript "object". .assert(condition, …) if "condition" is false, outputs a list of objects or a substitution string with objects. .count(label) outputs the number of times it has been called, with "label" if supplied, else with string "default". .time(label) starts a timer set to a "label" string that you can use to track how long an operation takes. .timeLog(label) logs to the console the current value of a timer. .timeEnd(label) stops and logs to the console the final value of a timer. JSON Parsing Methods JSON object with methods to handle conversion of JavaScript objects to and from JavaScript Object Notation strings. .parse(string) returns an object from the "string" parsed. .stringify(object) returns a JSON string from the "object" parsed.
